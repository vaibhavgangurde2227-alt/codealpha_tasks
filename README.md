Overview
This project was completed as part of the Data Analytics Internship. It covers web scraping, exploratory data analysis (EDA), and data visualization using real-world datasets. The objective was to collect structured data, perform analytical exploration, and generate meaningful insights through visual representation.

✅ Task 1 – Web Scraping

Scraped 50 pages from an e-commerce website.

Collected 1000 product records.

Extracted: Title, Category, Price, Rating, Availability.

Implemented pagination handling, relative URL correction, and request delays.

Exported structured dataset to CSV.


✅ Task 2 – Exploratory Data Analysis (EDA)

Performed data cleaning and preprocessing.

Handled missing values and duplicates.

Conducted statistical analysis and segmentation.

Generated insights using grouping and feature engineering.

✅ Task 3 – Data Visualization

Created bar charts, histograms, and pie charts.

Analyzed product distribution and customer segmentation.

Visualized trends and patterns for better decision-making insights.

🛠 Tools & Technologies

Python, Pandas, NumPy, BeautifulSoup, Requests, Matplotlib, Seaborn
